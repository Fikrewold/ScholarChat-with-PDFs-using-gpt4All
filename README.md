# 📚 Scholar Chat Bot with PDF 📚

## Overview
Scholar Chat Bot is an AI-powered application that allows users to interact with academic and research PDFs using natural language queries. By leveraging **GPT4All**, **FAISS** for vector storage, and **Streamlit** for the user interface, this tool helps users extract key insights from scientific documents efficiently.

## Features
✅ Upload multiple PDF documents (up to 200MB per file)  
✅ Extract and process text from PDFs using **PyPDF2**  
✅ Split text into manageable chunks for efficient processing  
✅ Store and retrieve embeddings using **FAISS**  
✅ Perform intelligent Q&A using **GPT4All**  
✅ Interactive web-based chat interface using **Streamlit**  

## Tech Stack
- **Programming Language:** Python  
- **Framework:** Streamlit  
- **AI Model:** GPT4All  
- **Vector Storage:** FAISS  
- **Text Processing:** PyPDF2, RecursiveCharacterTextSplitter  
- **Environment Management:** dotenv
